# Quick Reference Card: Add Booking to Your Website

## 🚀 Two Ways to Integrate

### Method 1: Direct Link (2 min)
**Perfect for**: Buttons, emails, social media

**Your Link:**
```
https://your-domain.com/book/your-org/centre
```

**Use it like this:**
```html
<a href="YOUR_LINK">Book Now</a>
```

---

### Method 2: Embed Code (5 min) ⭐ RECOMMENDED
**Perfect for**: Seamless website integration

**Step 1:** Copy the embed code from your dashboard  
**Step 2:** Paste it into your website's HTML  
**Step 3:** Test and publish  

---

## 📋 Platform Quick Guide

| Platform | How to Add |
|----------|------------|
| **WordPress** | Add "Custom HTML" block → Paste code |
| **Squarespace** | Add "Code" block → Paste code |
| **Wix** | Add "HTML iframe" → Paste code |
| **Shopify** | Edit page → "Show HTML" → Paste code |
| **Custom HTML** | Paste code where you want form |

## 🆘 Troubleshooting

**Form doesn't appear?**
→ Make sure you pasted the COMPLETE code
→ Use an HTML/Code block (not regular text)

**Form is cut off?**
→ Try the "Responsive" embed code
→ Or increase the height to 1000px
